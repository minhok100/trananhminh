# Student Management and Ranking System

This repository contains the final source code for the Student Management and Ranking System developed for Unit 19: Data Structures and Algorithms.

## Main features
- Add student
- Show ranking table
- Search student by ID
- Edit student
- Delete student

## Technologies
- Java
- Console application
- MaxHeap for ranking

## Repository purpose
This repository is attached as the final source code reference for the ASM submission.
